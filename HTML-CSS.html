What is HTML & CSS?

HTML Is an abbreviation for Hypertext Markup language. This is the standard and mainly used language when it comes to coding and creating websites. In class we broke down each of the abbreviations: HyperText is text that works with other text. Markup is the annotation of content to provide the relevant information (tags used in HTML) and finally Language which is basically the general syntax, and rules just like any other coding language. CSS stands for cascading style sheets. This is a language used for describing the presentation of a document written in a markup language. For example, in order to change the background color of a webpage that was created in HTML, CSS is required in order to specify what to change and in which style. There are also multiple ways to incorporate CSS code for a specific HTML document. This can either be through the use of external CSS which is basically coding CSS in a separate document and linking it to the HTML file or you can use Inline CSS with is basically coding CSS within the header of an HTML document. personally, I prefer the latter since It is easier to make changes quickly when the CSS is in the same file as your HTML code.
