What is Server Side Programming & Perl?

Server-side Scripting is used for web development that basically allows one to create scripts on a webpage. These scripts can produce specific responses based on what the users are doing on the site or what action a user has performed. Side Scripting can be done using a multitude of languages such as ASP, Active VFP, Java, PHP and Javascript and many more. when it comes to Scripting there are two main types that you may hear about. first, there is Client Side and Server side. Client-side scripting basically runs scripts on your computer after a specific website is loaded. Server-side is different in that scripts are not run on your computer but instead, they are run on specific servers that host the website that you are on. Perl is a language used for system administration, networking and other forms of server programming. It is commonly thought that Perl is a difficult language to learn but, in reality, it is pretty straightforward and actually uses a lot of features from other programming languages such as C and shell commands.
