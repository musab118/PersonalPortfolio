What is SQL?

SQL (Structured Query Language) is a standard for using relational database management systems. A relational database management system is one that manages given data and stores them into specific tables in a created database. SQL is a very widely used language and many companies use it solely because as a standardized relational language it has so many benefits. For one SQL has very low training costs. Second SQL can be a very productive tool when it comes to designing and implementing a database. SQL is also very portable and can be sustained for a long time.SQL also allows for cross-system communication meaning that one could technically manage a database from different computer systems. using SQL this semester has been very informative but there is still a lot to learn on my own personal time and it was definitely one of the more entertaining languages that we learned during the term.
