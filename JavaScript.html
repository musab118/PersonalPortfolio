What is JavaScript?

JavaScript (not to be confused with Java) is a lightweight programming language used for scripting HTML websites. Javascript can be used for many things. For example, it can be used to make web pages more interactive, It can be used to insert specific text into HTML such as text into boxes and usernames and passwords and contact information. JavaScript also allows a webpage to be able to respond to specific events. (For Example the navigation buttons!) JavaScript can also get information about a users computer and what browser that they are currently using as well as perform calculations on a users computer. JavaScript is also very important when performing form validation. This is basically when a webpage asks you to enter a phone number and JavaScript will validate that you have not entered any letters into the phone number field.
